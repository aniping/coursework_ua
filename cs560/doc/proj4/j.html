Join Algorithms

Three join methods are available in Minibase. Each join method is implemented as an iterator.

Join Algorithms:

  • Simple Nested Loops
  • Index Nested Loops
  • Sort Merge

See the text for a detailed description of the sort-merge (and other) join algorithms.
